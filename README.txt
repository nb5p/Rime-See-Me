『用途』
用於生成Rime配色方案，使用簡單，上手操作卽可。
生成配色方案適用於當前版本小狼毫。Linux與Mac均無測試紀錄。

『注意』
1、使用最新HTML5。Windows 7平臺測試，不兼容【當前版本Firefox】。請使用【Chrome內核】瀏覽器。IE未測試。
2、功能未完善。歡迎報Bug。

2、JSCOLOR項目
3、Chrome瀏覽器
